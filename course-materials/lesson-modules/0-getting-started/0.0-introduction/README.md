## About TaskFlow

Throughout this course, you'll work for **TaskFlow**, a fictional B2B SaaS company building project management software for remote teams.

Think "Asana meets Linear" - simple, fast, built for async-first companies.

**Your role:** Senior PM for activation & onboarding

All context is pre-created. You're stepping into a working company with:
- Company overview and goals
- User personas and research
- Product specs and competitive analysis
- Active projects and documents

---

## What is GitHub Copilot CLI?

Copilot CLI is a terminal-based AI assistant that can:
- Read and write files
- Run commands
- Search codebases
- Answer questions about your work
- Execute complex multi-step tasks

It's like having a smart assistant who can see your files and help you work through problems.
