## How to Use

1. Start in this directory (`course-materials/`)
2. Launch Copilot CLI: `copilot`
3. Follow the lessons in order
4. Save your work to the `outputs/` folder

## Quick Start

```bash
# Start Copilot CLI
copilot

# In Copilot, say:
> Let's start Module 1.1 - Welcome to the course!
```

## The TaskFlow Company

Throughout this course, you'll work for TaskFlow, a fictional B2B SaaS company. All company context is in `company-context/`.

Read:
- `taskflow-overview.md` - Company background and your role
- `taskflow-personas.md` - User personas (Sarah, Mike, Alex)
- `taskflow-product.md` - Product features and roadmap
- `taskflow-competitive.md` - Competitive landscape

## Tips

- Do modules in order (they build on each other)
- Actually do the exercises (don't just read)
- Save your outputs for reference
- Take breaks when needed
